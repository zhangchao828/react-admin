## 前端脚手架集成方案 react admin

## 安装

> 安装前需要先设置源

```shell
# npm
npm config set registry http://nexus.zswl.cn:8888/repository/npm-group

# yarn
yarn config set registry http://nexus.zswl.cn:8888/repository/npm-group

# install
npm install @zswl/cli -g
```

## 创建新工程

```shell
zswl create
```

## 启动项目

```shell
yarn start
## or
npm run start
```
